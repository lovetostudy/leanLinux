通过颜色判断文件的类型是不一定正确的！！！
Linux系统中文件是没有扩展名！！！(扩展名的意义在于给人看)


-	普通文件(文本文件，二进制文件,压缩文件,电影,图片...)
d	目录文件
b	设备文件(块文件(block))存储设备硬盘,U盘 /dev/sda, /dev/sda1
c	设备文件(字符设备)打印机,终端	/dev/tty1, /dev/zero
s 	套接字文件
p	管道文件
l	链接文件


$ file /etc/host		查看文件类型的命令


-------------------------------------------------------------------------------
type	查看命令类型，例如该命令是alias,还是内置命令，还是某个文件

stat	文件的属性，例如文件的名称，大小，atime,ctime.......

file	查看文件类型  例如，文本文件，二进制文件。。。。
